GUID

Any request to the API must be done by providing a valid token plus your api key. The token can be retrieved through an authentication request. The authentication request itself must be done by providing a valid API key.

Authentication

Authenticate

When successfully authenticated, you are given a token which is valid for a maximum of 1 day.

REQUEST

GET

HOST/authenticate
Headers
Content-Type application/json
x-api-key api_key_value

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "status": "authenticated",
    "value": "0251791b002f142fe7524f5770b63f78d6a719471a690a17955e31df7fe333db",
    "expires": "2018-03-27 08:57:09"
}
GUIDS Object

Get GUID object

Retrieve information about a guid object

REQUEST

GET

HOST/guids/{guid}
Parameters
guid string

GUID value
Headers
Content-Type application/json
x-api-key api_key_value
x-token token_value

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "data": {
        "guid": "e0f1b2c8-d48f-4141-be90-cd05679083e6",
        "status": "assigned",
        "assigned_to": "EMMA",
        "created_at": "2018-03-26 09:09:20",
        "assigned_at": "2018-03-27 10:19:25"
    }
}
